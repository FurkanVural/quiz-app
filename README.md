# Quiz App

HTML, CSS ve JavaScript kullanılarak geliştirilmiş basit ve etkileşimli bir quiz uygulaması.  
Kullanıcı quiz’i başlatabilir, soruları süreye karşı cevaplayabilir, doğru/yanlış geri bildirimi alabilir ve test sonunda skorunu görebilir.

## Proje Hakkında

Bu proje, temel JavaScript mantığını, DOM manipülasyonunu ve kullanıcı etkileşimini pratik etmek amacıyla geliştirilmiştir.  
Uygulama içerisinde soru yönetimi, kullanıcı arayüzü kontrolü, süre mekanizması ve skor hesaplama gibi temel yapılar bulunmaktadır.

## Özellikler

- Quiz başlatma ekranı
- Çoktan seçmeli soru yapısı
- Her soru için geri sayım süresi
- Süreye bağlı ilerleme çizgisi
- Doğru / yanlış cevap kontrolü
- Sonuç ekranında skor gösterimi
- Quiz’i yeniden başlatma ve çıkış yapma butonları
- Modüler JavaScript dosya yapısı

## Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript (Vanilla JS)
